Drink with a Straw.

Don't swallow the ocean. Standard libraries try to load the entire 8GB file into RAM, killing your process immediately.

Molniya's scanCsv creates a zero-copy stream, allowing you to process terabytes of data on a micro-instance without ever hitting the memory ceiling.

Type-Safe API

Built natively for TypeScript and Bun. Enjoy full Intellisense and types that actually work.

  • Zero-copy chunking
  • Lazy evaluation
  • Panic-free error handling
analysis.ts 
import { scanCsv } from 'molniya';

// 1. Create a Lazy Stream (No heavy loading)
const stream = await scanCsv('transactions_2025.csv', { 
  lazy: true,
  batchSize: 50_000 
});

// 2. Process in-flight
for await (const batch of stream) {
  batch
    .filter(row => row.amount > 100)
    .groupBy('currency')
    .print(); 
  
  // 3. Memory is instantly freed after loop
}
